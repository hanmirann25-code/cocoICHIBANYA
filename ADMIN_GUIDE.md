## 📝 공지사항 관리

### 1️⃣ 공지사항 목록 보기
로그인 후 자동으로 공지사항 목록이 표시됩니다.

**표시 정보:**
- 번호
- 카테고리 (공지/이벤트/시스템)
- 중요 표시
- 제목
- 작성일
- 관리 버튼 (수정/삭제)

---

### 2️⃣ 새 공지사항 작성

1. **"새 공지사항 작성" 버튼 클릭**

2. **정보 입력:**
   - **제목** (필수): 공지사항 제목
   - **카테고리** (필수): 공지, 이벤트, 시스템 중 선택
   - **작성자**: 기본값 "코코이찌방야"
   - **중요 공지사항**: 체크하면 상단에 고정 표시
   - **내용** (필수): 공지사항 본문

3. **미리보기 확인**
   - 하단에서 실시간으로 미리보기 확인 가능
   - 줄바꿈, 서식 등 확인

4. **저장 버튼 클릭**

**💡 작성 팁:**
- 줄바꿈은 Enter로 입력하면 자동 반영
- 중요한 공지는 "중요 공지사항" 체크
- 이벤트성 공지는 카테고리를 "이벤트"로 선택

---

### 3️⃣ 공지사항 수정

1. 목록에서 **"수정" 버튼 클릭**
2. 내용 수정
3. 미리보기 확인
4. **"저장" 버튼 클릭**

---

### 4️⃣ 공지사항 삭제

1. 목록에서 **"삭제" 버튼 클릭**
2. 확인 메시지에서 **"확인"**
3. 영구 삭제 (복구 불가)

---

### 실서버 전환 방법

#### 📍 단계 1: 백엔드 API 준비

**필요한 API 엔드포인트:**
```
GET    /api/notices          # 전체 공지사항 조회
GET    /api/notices/:id      # 특정 공지사항 조회
POST   /api/notices          # 공지사항 생성
PUT    /api/notices/:id      # 공지사항 수정
DELETE /api/notices/:id      # 공지사항 삭제
```

**데이터베이스 스키마 예시:**
```sql
CREATE TABLE notices (
  id INT AUTO_INCREMENT PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  category VARCHAR(50) NOT NULL,
  content TEXT NOT NULL,
  isImportant BOOLEAN DEFAULT FALSE,
  author VARCHAR(100),
  date DATE,
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### Q1. 관리자 비밀번호를 변경하려면?
**A:** `src/pages/AdminNoticePage.jsx` 파일에서:
```javascript

```

### Q2. 카테고리를 추가하려면?
**A:** `src/pages/AdminNoticePage.jsx`의 Select 부분에 추가:
```javascript
<option value="신규카테고리">신규카테고리</option>
```

**A:** LocalStorage 방식의 한계입니다.
- 브라우저 캐시 삭제 시 소실
- 실서버 API 방식으로 변경 권장

**A:** LocalStorage는 브라우저마다 별도 저장됩니다.
- 실서버 API 방식으로 변경 필요

### Q5. 이미지를 추가하고 싶어요!
**A:** 현재는 텍스트만 지원합니다.
- 향후 업데이트 예정

## 💬 5. 창업 상담 관리

### 📌 개요
사용자가 창업 안내 페이지에서 신청한 상담 내역을 조회하고 관리합니다.

### 🔗 접속 방법
```
URL: http://localhost:5173/#/admin/consultation
또는 관리자 대시보드에서 "창업 상담 관리" 클릭
```

---

### 1️⃣ 상담 신청 목록 보기

**표시 정보:**
- 신청자 이름
- 연락처 (전화번호, 이메일)
- 희망 지역
- 문의 내용 (미리보기)
- 신청일시
- 처리 상태 (대기중/연락완료/처리완료)

**상태별 통계:**
- 🕐 대기중: 아직 연락하지 않은 신청
- 📞 연락완료: 연락은 했으나 처리 진행 중
- ✅ 처리완료: 상담 완료

---

### 2️⃣ 상담 신청 상세 보기

1. **"상세보기" 버튼 클릭**

2. **확인 가능한 정보:**
   - 신청자 이름
   - 연락처 (전화번호)
   - 이메일
   - 희망 지역
   - 문의 내용 (전체)
   - 신청일시

3. **처리 상태 변경:**
   - 드롭다운에서 상태 선택
   - 자동 저장

4. **관리자 메모 작성:**
   - 통화 내용, 처리 내역 등 기록
   - "메모 저장" 버튼 클릭

---

### 3️⃣ 상담 신청 삭제

1. **목록에서 휴지통 아이콘 클릭**
2. **확인 메시지에서 "확인"**
3. **삭제 완료**

⚠️ **주의**: 삭제된 데이터는 복구할 수 없습니다!

---

### 💡 상담 관리 팁

**효율적인 관리:**
1. 매일 아침 "대기중" 상태 확인
2. 연락 후 즉시 "연락완료"로 변경
3. 관리자 메모에 통화 내용 기록
4. 처리 완료 후 "처리완료"로 변경

**메모 작성 예시:**
```
2026-01-26 10:30 통화
- 서울 강남구 희망
- 투자 가능 금액: 1.5억
- 3월 오픈 희망
- 다음주 화요일 본사 방문 예정
```
